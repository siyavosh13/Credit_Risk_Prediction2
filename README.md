پیش‌بینی اعتبار مشتریان برای دریافت وام بانکی با یادگیری ماشین
این پروژه یک سیستم هوشمند برای پیش‌بینی ریسک اعتباری مشتریان است؛ یعنی کمک می‌کند بانک یا مؤسسات مالی تشخیص دهند که آیا یک مشتری واجد شرایط دریافت وام هست یا خیر.
این مدل با استفاده از داده‌های واقعی شامل اطلاعات مالی، سوابق اعتباری، درآمد، سابقه پرداخت و … آموزش دیده و توانایی ارزیابی ریسک هر متقاضی را دارد.
