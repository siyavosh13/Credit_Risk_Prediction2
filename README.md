# pardakhtvam
پیش بینی اعتبار وام دهی 
 این پروژه با ماشین لرنینگ ساخته شده و توی این پروژه براساس دیتاستی که داریم و اطلاعات افراد وام گیرنده مثل شغل مقدار درامد و مابقی فاکتورهایی که توی دیتاست هستند پیش بینی میکنه که درصد خوش حسابی فرد و پرداخت اقساط وامش چقدر هست این یک پروژه کاربردی برای بانک هاست برای اینکه تعلق گرفتن وام به مشتری هارو بسنجن
